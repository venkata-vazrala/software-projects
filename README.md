# Software Projects

An umbrella index of standalone software applications and backends. Each is its own repo, linked here as a submodule.

| Project | Description |
|---|---|
| [Intern_day_apple](https://github.com/venkata-vazrala/intern_day_apple) | Automation Build & Test Tracker — a Django + DRF backend that tracks multi-team Build → Test → Deploy pipelines with weighted scoring and trend metrics (built for an Apple internship interview) |
| [soliton-internship](https://github.com/venkata-vazrala/soliton-internship) | Internship work at Soliton Technologies — a Python/Flask web-forecast REST API and a companion Python CLI (each linked as a nested submodule) |

## Clone with all submodules

```bash
git clone --recurse-submodules https://github.com/venkata-vazrala/software-projects.git
# or, if already cloned:
git submodule update --init --recursive
```
